# Hotel Management Web UI

This is a simple hotel management system built using HTML, CSS, and JavaScript.

## Features

- View room details
- Check room availability
- Book rooms (UI only)
- Order food
- Checkout summary

## How to Run

1. Open `index.html` in your browser
2. Or run using Live Server in VS Code
